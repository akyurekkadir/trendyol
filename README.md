# 🍏 macOS Portfolio: Kadir Akyürek

Welcome to my interactive, macOS-inspired personal portfolio! This project is a full-screen React application that simulates the macOS desktop environment, featuring a rich graphical user interface, draggable windows, and smooth animations.



## ✨ Features
- **Interactive Apps**: Experience custom-built applications including **Terminal**, **Finder**, **Safari**, **Launchpad**, **Resume**, and a specialized **Trendyol App**.
- **Trendyol Showcase**: A fully customized environment showcasing my career timeline via an advanced SVG tracking system, datathon awards, technical skills, and a "cart" system for my Talent Program application.
- **Draggable Windows**: Fully draggable, minimal, and resizable window components powered by `react-rnd`.
- **Responsive Animations**: Beautiful transitions, drop shadows, and visual fidelity matching the standard macOS aesthetic.

## 🛠️ Built With
- **React.js (Vite)**
- **Tailwind CSS v4**
- **Lucide React**

## 📦 How to Run Locally
To run this application on your local machine:

```bash
# Clone the repository
git clone https://github.com/akyurekkadir/trendyol.git

# Navigate into the project directory
cd trendyol

# Install dependencies
npm install

# Start the dev server
npm run dev
```
